# 劇集名稱重組與標準化工具 (Rename Tool v2)

一款專為劇集、動漫與影音檔案設計的名稱標準化與批次重新命名 GUI 工具。本工具採用現代感的 **Catppuccin Mocha** 暗色系主題設計，提供直觀的雙欄操作介面（側邊欄設定 + 右側即時對照表），幫助您快速清理並統一影片檔名的雜贅資訊。

---

## 🌟 核心特色

- **🎨 現代化精美介面**
  - 基於 Tkinter 深度客製化的 Catppuccin Mocha 暗色調主題。
  - 支援按鈕懸停（Hover）動態色彩變換。
  - 輸入框聚焦（Focus）高亮藍色邊框提示。
- **🔍 智慧集數識別 (Regex Matching)**
  - 自動偵測並提取各種複雜檔名中的集數資訊。
  - 支援解析常見格式如：`S01E05`、`E12`、`EP03`，或是檔名中被符號/空格隔開的純集數數字。
- **⚙️ 自訂標準化結構**
  - **標題一**：通常輸入劇集名稱（例如：`凡人修仙傳`）。
  - **標題二**：可用於畫質或來源標籤（例如：`1080P` 或 `NF`）。
  - **設定季數**：統一轉換為標準格式（例如：`S01`、`S02` 等）。
  - 自動重組為：`[標題一].[標題二].[季數]E[雙位數集數].[原副檔名]`。
- **👁️ 即時安全預覽**
  - 掃描後於右側 Treeview 呈現「原檔名」與「標準化新檔名」雙欄對照。
  - 執行前會跳出二次彈窗確認，避免誤觸修改。
- **💾 自動記憶配置**
  - 可選擇開啟「自動儲存設定欄位」，關閉程式或執行掃描時，將自動把「資料夾路徑、標題一、標題二、季數」寫入 `rename_config.json`，下次啟動自動帶入。
- **📦 執行檔一鍵打包**
  - 隨附 PyInstaller 的 `.spec` 設定檔，可直接打包成無終端機黑視窗的獨立 `.exe` 檔案。

## 🛠️ 開發環境與需求

- **作業系統**：Windows / macOS / Linux (本工具已在 Windows 系統進行 GUI 優化)
- **Python 版本**：Python 3.x
- **依賴套件**：僅使用 Python 內建的標準庫（`tkinter`, `os`, `re`, `json`），**免額外安裝 `pip` 套件**，開箱即用。

## 🚀 快速開始

### 1. 直接執行原始碼
確認系統已安裝 Python 3，於終端機執行：
```bash
python rename_tool_v2.py
```

### 2. 欄位填寫說明
1. **劇集資料夾路徑**：點選「瀏覽資料夾」選擇您要重新命名的影片目錄。
2. **標題一**：設定主要劇名。
3. **標題二**：設定次要標籤（如不需要可留空，重組時會自動忽略空白欄位）。
4. **設定指定季數**：輸入如 `S01` 或 `S02`（程式會自動強制轉換為大寫）。
5. **自動儲存設定欄位**：勾選後將在每次掃描或關閉視窗時將欄位儲存至 `rename_config.json`。

### 3. 操作流程
```
[選擇資料夾與設定欄位] ➜ 點擊 [掃描與分析檔案] ➜ [於右側預覽確認對照] ➜ 點擊 [執行更名] ➜ [確認彈窗] ➜ 完成！
```

## 📝 規則與邏輯說明

### 1. 支援的影片副檔名
程式僅會掃描並修改以下副檔名的影片：
`.mp4`, `.mkv`, `.avi`, `.mov`, `.wmv`, `.flv`, `.rmvb`, `.ts`

### 2. 集數識別正規表示式 (Regex)
系統使用以下正則表達式來定位集數：
```regex
[Ss]\d{1,2}[Ee](\d{1,2})|[Ee][Pp]?(\d{1,2})|[^a-zA-Z0-9](\d{1,2})[^a-zA-Z0-9]
```
- **比對優先權一**：`S01E05` ➜ 提取 `05`
- **比對優先權二**：`E12` 或 `EP03` ➜ 提取 `12` 或 `03`
- **比對優先權三**：`[08]`、`- 08 -` 或 `_08_` 等被非英數字元包夾的數字 ➜ 提取 `08`
- 提取到的集數將自動格式化為雙位數（如 `1` 轉為 `01`，`12` 保持 `12`）。

## 📦 打包成獨立 Executable (.exe)

本專案已包含 `rename_tool_v2.spec` 設定檔。若您想將其打包成免安裝的 `.exe` 執行檔，請依照以下步驟操作：

1. 安裝 PyInstaller：
   ```bash
   pip install pyinstaller
   ```
2. 使用 spec 檔案進行打包：
   ```bash
   pyinstaller rename_tool_v2.spec
   ```
3. 打包完成後，可以在產生的 `dist/` 資料夾下找到 `rename_tool_v2.exe`。
